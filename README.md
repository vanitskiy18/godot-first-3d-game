# 🤖 Godot First 3D Game
![Godot](https://img.shields.io/badge/Godot-3.5-blue)
![Status](https://img.shields.io/badge/status-learning_project-green)

A simple **3D tutorial game** made in **Godot 3.5**.  
This project was created as an introduction to **3D game development in Godot**.

---

## 🎮 Gameplay

The goal of the game is to **collect all coins on the level** while avoiding enemy robots.

Robots patrol the area and attack the player if they get close.

You can defeat robots by **shooting them and pushing them off the platforms**.

---

## 🎯 Objective

To complete the level the player must:

1. Collect **all coins**
2. Survive enemy robots

---

## 🎮 Controls

| Key | Action |
|----|----|
| **W A S D** | Move |
| **Space** | Jump |
| **Right Mouse Button** | Shoot |

---

## ⚙️ Game Mechanics

- The player has **3 HP**
- A robot attack removes **1 HP**
- Falling off the map removes **1 HP**
- The level is completed when **all coins are collected**

---

## 🤖 Enemies

Robots move around the level and try to stop the player.

They can be defeated by **shooting them and knocking them off the platforms**.
